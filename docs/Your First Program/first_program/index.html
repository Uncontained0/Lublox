Your First Program

This tutorial will show you how to require the module, and create a client.

Text Editorâ€‹

The first important step to development of any kind is to pick your text editor. Any editor that can edit simple text files will work, but things like notepad might not work for you. I personally use Visual Studio Code, but I would also recommend Atom and Sublime.

Loading the Moduleâ€‹

Make your entry file, I suggest main.lua. Require the module and create a client like in the code below.

local Lublox = require("Lublox")
local Client = Lublox.Client ()

The client will be used to make all requests and construct all objects. Lets get the "About Me" section of the roblox profile.

local User = Client:User ("Roblox")

print(User.Description)

We first create the User object, then print the Description property. To run your code open a terminal of your choice, navigate to the file, and run the file with luvit.

luvit main.lua
